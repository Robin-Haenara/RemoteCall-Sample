# RemoteCall-Sample
📞 Remote call sample android app using RemoteMonster

## How to setup

Service ID와 Key는 아래링크에서 발급 받을 수 있습니다.
[https://connectlive.kakaoi.ai/](https://connectlive.kakaoi.ai/)

`local.properties`파일에 발급받은 Service ID와 Key를 아래와 같이 추가 해야합니다.

```properties
service_id=...
api_key=...
```
